> 对用户的情绪进行分析，并给出可行的调整方案

![演示截图](demo.gif)


## 💡项目简介
**在这个网页中，我们将对您的情绪进行分析，并给出具体可行的调整方案，希望在面临DDL和各种考验的过程中，它能对您的情绪调控有所帮助，助您过关斩将；在平淡无聊的日常琐事中，它也能给您提供一些可能性，让您打破日常的循环，走出新的路径，探索那些生活中未知的、微小的美好。**


## 🌟 核心功能
- 智能情绪检测
- 基于用户情绪给出具体可行的调整方案
